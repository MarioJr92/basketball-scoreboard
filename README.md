# Basketball Scoreboard

This is a simple basketball scoreboard created using HTML, CSS, and JavaScript. It allows users to keep track of the score of a basketball game by incrementing the score of the home and away teams.

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository to your local machine
2. Open the `index.html` file in your browser
3. Start keeping track of the score of your basketball game!

## Usage

To increase the score of the home team, click on the "+1", "+2", or "+3" buttons. Similarly, to increase the score of the away team, click on the corresponding buttons in the away team section.
